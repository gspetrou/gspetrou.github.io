Prone Mod

A videogame addon that adds crawling capabilities to the game Garry's Mod.

  • Uses custom player animations and hand-crafted aimlayers for fluid movement.
  • Built on peer-reviewed netcode to ensure all hitboxes remain accurate.
  • Runs off an optimized animation handler to ensure little to no frame drops.
  • Multiple versions available to support every gamemode and most popular animation frameworks.

Smart Weld Tool

A tool used in the videogame Garry's Mod to boost building efficiency and stability.

  • Can intelligently "smart select" nearby items that you may want to weld together.
