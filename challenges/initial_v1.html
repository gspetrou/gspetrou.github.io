Iterative, In-Order, Binary Search

Problem: Determine whether a given binary tree is valid.
Hint: Use a stack.

Solution: This function checks the validity of a binary tree using an iterative, in-order, binary search algorithm.

def isValidBST(self, root: TreeNode) -> bool:
	nodeList = []
	stack = []
	pre = None

	while root or stack:
		while root:
			stack.append(root)
			root = root.left

		root = stack.pop()
		nodeList.append(root.val)
		root = root.right
	return nodeList

Remove element, in place, from list

Problem: Given list and number, remove all occurrences of number from that list in place.
Hint: This problem is interesting because it uses two left pointers.

Solution: Left pointer simply loops through the list. Left pointer 2, the length variable, increases when we find a number that we don’t want to remove.

def removeElement(self, nums: List[int], val: int) -> int:
	if not nums:
		return 0

	length = 0
	for v in nums:
		if v != val:
			nums[length] = v
			length += 1
	return length

Remove duplicates of sorted array in place

Problem: Remove duplicate numbers of sorted array in place.
Hint: This problem can be solved either with a left and right pointer (slower), or two left pointers (faster).

Solution 1 — Left and Right Pointers (Slower):
The left pointer is incremented when a non-target is found, just advance, no worries. When a target value is found then swap values at each pointer and decrement right.

def removeElement(self, nums: List[int], val: int) -> int:
	if not nums:
		return 0

	left = 0
	right = len(nums) - 1
	while left <= right:
		if nums[left] == val:
			nums[left], nums[right] = nums[right], nums[left]
			right -= 1
		else:
			left += 1

	return left

Solution 2 — Two Left Pointers (Faster): Both pointers start at beginning, one increases every value. The other only increases when an element that needs removing comes by to fill up the array with proper values starting from the beginning.

def removeElement(self, nums: List[int], val: int) -> int:
	if not nums:
		return 0

	length = 0
	for v in nums:
		if v != val:
			nums[length] = v
			length += 1
	return length

Maximum Subarry (Kadane’s Algorithm)

Problem: Given an array, find the range in that array which sums up to the largest value.
Hint: Can be done in O(n) time using dynamic programming.
Hint 2: dp[i] = max(dp[i-1]) + arr[i] ∀ i ∈ [1, n-1]

Solution (Kadane’s Algorith):

def maxSubArray(self, nums: List[int]) -> int:
	for i in range(1, len(nums)):
		if nums[i-1] > 0:
			nums[i] += nums[i-1]
	return max(nums)

Two Sum

Problem: Given an array and a target find the two values that add up to target.
Hint: Using a map, key = value, value = index

Solution:

def twoSum(self, nums: List[int], target: int) -> List[int]:
	numDict = {}

	for i, val in enumerate(nums):
		if val in numDict:
			return [numDict[val], i]
		numDict[target - val] = i
	return -1
